This repository contains the source code for my personal website, a Flask-based application with a dynamic frontend. The website is designed with a mix of HTML, CSS (utilizing SCSS for styles preprocessing), and Python.

## Features

- **Flask Backend:** Lightweight web server with Python for easy scalability and maintenance.
- **SCSS Styling:** Advanced styling with SCSS for better structure and design customization.
- **Responsive Design:** Fully responsive HTML and CSS to ensure the site looks great on all devices.
